# 牍

**牍（du）**，改良尚书体之器也。启之，Claude Code 即以**改良尚书体**应子——古为密度之用，省 token 而人可读、模型不滑落。**尚书之骨，金石之简。**

姊妹插件 [`wen`](https://github.com/F2077/wen)（文）：文者，对话之古辞，主审美；牍者，记事之古简，主密度。一文一牍，一语一记，一散一密。

## 置

```
/plugin marketplace add https://github.com/F2077/du
/plugin install du@du-marketplace
```

置毕即行，毋需他设。欲止之：

```
/plugin disable du
```

## 道

以 `SessionStart` hook 于每会话之初注牍式于上下文，**每会话一注，后不复加**。**唯改辞气，不涉工程**——commit、注释、文档诸落盘之物，咸用平实中文或英文，不为牍所染。

牍以尚书之骨为根：省主语、一事一记、状态前置、助字分工。砍其戏仿腔（王若曰/嗟/用是）与古奥字，缩至先秦常用字，设状态词闭集。**条件省**者，其核心也——主语可推则省，不可推则补——故工作流极密，日常对话遇歧自补。详参 [`牍式.md`](references/牍式.md)。

## 铭

遇真里程碑——大 feature 落成、大 bug 克复、大重构毕——牍可缀**铭**以记（唯对话，非落盘）。铭者，古之金石记功之辞，四言为体，劲峭简古，至多两句。不效 [`wen`](https://github.com/F2077/wen) 之吟诗作赋——赋主铺陈，与牍之密相违；铭主凝练，与牍之密同根，且金文铭辞本即牍之古源。闲常勿铭，疑则不铭。

## 约

同一意，白话与牍之别（CI/CD 状态示例）：

| | 字数 | 折 token（约）|
|---|---|---|
| 白话「代码已经拉下来了。依赖装好了，没有冲突。构建跑起来了，还在进行。单元测试通过了，43 个全过。集成测试失败了，超时，可能是数据库连接池太小。正在查。部署没开始，等审批。」 | ~80 | ~55 |
| 牍「拉码毕。依赖装，无虞。构建起，行中。单测尽过，计四十有三。集测败，超时，疑连池隘。方察。部署待，候决。」 | ~38 | ~26 |

牍之令约 4,300 字符，折 token 约 1,100。
