遇真里程碑——大 feature 落成、大 bug 克復、大重構畢——牍可綴銘以記。 四言為體，勁峭簡古，至多兩句。不效文之吟詩作賦；銘主凝練，與牍之密同根，且金文銘辭本即牍之古源。閒常勿銘，疑則不銘。

條件省 + 三道防——密度隨場景自適應，無須切換檔位。

一
硬條件

省主語，唯前句同主、或主為當前之務乃可；不然，必出。

二
首句主出

每段首句主語必出，後續承上省——給讀者一主語錨點。

三
歧義即補

省後一句可作兩解者，寧補毋省。密度讓位於不歧。

子子孫孫永寶用
——《毛公鼎》嘏辭
